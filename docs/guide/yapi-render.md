# YApi Render

EasyYapi supports custom rendering when exporting to YApi.

## Render Mode

When exporting APIs to YApi, EasyYapi can render the API documentation with different modes:

- **Smart render**: Automatically merge and update existing APIs
- **Overwrite render**: Overwrite all existing APIs in the category

## Configuration

Configure the render mode in the YApi settings or in the configuration file.

See [YApi Settings](/settings/yapi) for more details.
