# Legacy YApi / Custom YApi

If you are using a customized version of YApi or an older version, you may need additional configuration.

## Login Mode

EasyYapi supports different login modes for YApi:

- **Token mode**: Use project token for authentication (recommended)
- **Account mode**: Use username/password for authentication

## Configuration

Configure the login mode in the YApi settings:

1. Open <kbd>Settings</kbd> > <kbd>Other Settings</kbd> > <kbd>EasyApi</kbd>
2. Select the YApi login mode
3. Enter the required credentials

See [YApi Settings](/settings/yapi) for more details.
