# 传统 YApi / 自定义 YApi

如果你使用的是定制版的 YApi 或较旧的版本，可能需要进行额外的配置。

## 登录模式

EasyYapi 支持 YApi 的不同登录模式：

- **Token 模式**：使用项目令牌进行认证（推荐）
- **账号模式**：使用用户名/密码进行认证

## 配置

在 YApi 设置中配置登录模式：

1. 打开 <kbd>Settings</kbd> > <kbd>Other Settings</kbd> > <kbd>EasyApi</kbd>
2. 选择 YApi 登录模式
3. 输入所需的凭据

详见 [YApi 设置](/zh/settings/yapi)。
