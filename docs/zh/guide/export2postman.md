# 导出到 Postman

## 使用方法

### 方式一：右键导出

1. 打开项目中包含 API 的文件，或者在 IDEA 左边项目文件区域选择文件或文件夹
2. 鼠标右键点击文件内容或文件夹
3. 选择 `ExportPostman` 导出该文件或文件夹中所有的 API

### 方式二：快捷键导出

1. 打开项目中包含 API 的文件，或者在 IDEA 左边项目文件区域选择文件或文件夹
2. 使用快捷键 `Alt + Shift + E` (Windows/Linux) 或 `Ctrl + E` (macOS)
3. 选择要导出的 API，选择导出渠道 `Postman`
4. 点击 ✔ 按钮或按回车键完成导出
